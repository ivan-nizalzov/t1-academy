 # T1 Academy

----

### A classic REST-like app with in-memory repository, Spring Boot 3.2.0, and Apache Kafka.

----
### Tech Stack::
1. Java 17;
2. Spring Boot 3.2;
3. Lombok
4. Maven;
5. Jackson;
6. Mockito;
7. Docker;
8. Apache Kafka.

----
### How to run:
1. `mvn clean package`
2. `docker compose up`
